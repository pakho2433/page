# Mori Select Online Store

A responsive Next.js storefront designed for deployment on Vercel.

## Features

- Responsive design for phone, tablet, and desktop
- Product catalogue
- Shopping cart with quantity controls
- Automatic subtotal calculation
- WhatsApp order generation
- Newsletter and contact sections

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Import this GitHub repository into Vercel.
2. Keep the framework preset as `Next.js`.
3. Click **Deploy**.

Vercel will automatically detect the build command and output settings.
